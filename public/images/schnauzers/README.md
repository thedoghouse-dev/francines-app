# Schnauzer Gallery Images

## Add Your Photos Here

Place your schnauzer photos in this folder with these exact names:

1. `schnauzer-1.jpg` - Classic Schnauzer Portrait
2. `schnauzer-2.jpg` - Playful Schnauzer
3. `schnauzer-3.jpg` - Alert Watchdog
4. `schnauzer-4.jpg` - Loyal Companion
5. `schnauzer-5.jpg` - Intelligent Friend
6. `schnauzer-6.jpg` - Distinguished Gentleman

## Image Specifications

**Recommended:**
- **Size**: 1200 x 900 pixels (4:3 aspect ratio)
- **Format**: JPG or PNG
- **File Size**: 200-300 KB per image (max 500 KB)

**Notes:**
- Images will automatically resize to fit the screen
- If an image is missing, a placeholder will be shown
- The gallery will still work with any size images, but 4:3 ratio looks best

## After Adding Images

1. Rebuild: `npm run build`
2. Upload the new `build/` folder to your server via FTP
3. Your photos will appear in the Schnauzer Gallery!
